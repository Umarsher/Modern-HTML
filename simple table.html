HTML vs HTML5

HTML HTML5
HTML is the previous version of HTML5. HTML5 is the current version of HTML.
HTML has limited support for multimedia elements, requiring plugins like Flash. HTML5 provides native support for multimedia elements like <audio> and <video>, eliminating the need for plugins.
HTML does not have dedicated semantic elements. HTML5 introduced new semantic elements like <header>, <nav>, <section>, and <article> for better document structure and semantics.
